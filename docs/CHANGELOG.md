# Changelog

## v0.5.0

### Added

- WebRTC DataChannel による軽量通信対戦を追加
- 手動オファー/回答コード交換UIを追加
- ホスト先手 / ゲスト後手の役割管理を追加
- 接続直後の局面同期を追加
- 指し手同期、投了同期、局面再送を追加
- 通信指し手の手数・手番・合法手チェックを追加
- 通信プロトコルの自動テストを追加

### Changed

- 通信中はルールセット変更、初期化、1手戻す、読込、棋譜再生を無効化
- `npm test` は TAP レポーターと `--test-force-exit` を使用

## v0.4.0

### Added

- 11x11拡張検証ルールセットを追加
- 追加駒「麒麟」「銅将」「走兵」を追加
- ルールセット切り替えUIを追加
- 拡張盤面と追加駒の自動テストを追加

## v0.3.0

### Added

- 棋譜ログ表示を追加
- 1手戻す機能を追加
- 棋譜再生機能を追加
- JSON書き出し / JSON読み込みを追加
- localStorage保存 / 読み込みを追加
- 棋譜から局面を復元するreplay機構を追加
