## v1.1.0 - UI Phase 1

- 盤面をページ上部へ移動し、対局中の視認性を改善。
- 対局設定、持ち時間、通信、メタ情報、操作説明を初期折りたたみに変更。
- 盤面直上に手番バーを追加し、手番・手数・通信状態・王手を見やすく表示。
- 手番側の持ち駒欄と盤上駒を軽く強調。

## v1.2.0 - UI改善 Phase 2

- 成り確認を `window.confirm()` から画面内の成り選択UIへ変更。
- 棋譜ログを折りたたみ可能に変更し、手数バッジを追加。
- 手番バーに「次は第n手」を表示。
- 対局中に使う主要ボタンの視認性を改善。
- スマホ幅で棋譜ログと成り選択UIが扱いやすいようCSSを調整。

## v1.3.0 - UI改善 Phase 3

- 盤面視点を先手視点 / 後手視点で切り替え可能にした。
- 最後の指し手の移動元・移動先を盤面上で強調表示した。
- 王手中の玉を盤面上で強調表示した。
- 合法手ハイライトのON/OFFを追加した。
- 投了・初期化前の確認ON/OFFを追加した。
- 表示と操作確認の設定パネルを盤面直上に追加した。
